环境配置

  • github 共钥设置

  • 到github添加 cat ~/.ssh/id_rsa.pub 得到的公钥设置码
  • github个人账号设置下面，添加ssh
    • 上传3步曲
      • git add
    • git commit -am “tagname”
    • git push

    • hexo s
